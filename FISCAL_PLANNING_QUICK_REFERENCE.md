## Common Tasks

### Generate Strategies for One Priority
1. Click "Generate AI Strategies"
2. Select only one priority checkbox
3. Add context about that priority
4. Click "Generate"

### Convert Only High-Priority Strategies
1. Wait for blue "Convert" banner
2. Click "Convert to OGSM"
3. Uncheck low-priority strategies in modal
4. Convert only selected ones

### Track Strategy with KPIs
1. Find approved strategy
2. Click "Create KPIs"
3. Review auto-populated metrics
4. Add targets and units
5. Click "Create Selected KPIs"

### Edit Priorities After Creation
1. Go to Plan Detail
2. Click priority card
3. Edit title/description in Setup page

## Troubleshooting Quick Fixes

| Problem | Quick Fix |
|---------|-----------|
| Can't activate plan | Convert all approved strategies first |
| No blue banner | Approve at least one strategy |
| "Add to Plan" fails | Ensure plan is in draft status |
| Strategies won't generate | Add more context to AI prompt |
| KPI creation failed | Ensure KPI has a name |
